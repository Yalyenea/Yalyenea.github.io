AI / Agent / 编程工作流

Manggo：Pot 不再维护后，作者推出的新翻译工具。原 Pot 仍可作为迁移背景参考。

opendatalab/Sciverse-Agent-Tools：面向 Sciverse Open Platform 的 agent 工具 schema 与 SDK，把检索能力整理成 LLM agent 可调用的标准工具。

Terax：terminal-first 的 AI-native dev workspace，偏向把 AI 开发工作流放回终端环境。

Gitlawb/zero：强调本地、自主、可控的 coding agent，口号是你的模型、机器和规则由你决定。

Tiktokenizer：在线 tokenizer 观察工具，适合快速看文本如何被 token 化。

偷懒工具：一组在线小工具集合，偏日常效率场景。

UniClipboard：local-first、P2P、端到端加密的跨设备剪贴板同步工具，不依赖账号和云端中转。

机器学习 / 优化 / 课程

Random Reshuffling Dominates Stochastic Gradient Descent：证明在光滑凸优化中，合理步长下 random reshuffling 在任意有限 epoch 后都优于 SGD，回答了一个长期开放问题。

Orca: The World is in Your Mind：world model 方向论文页面，关注模型内部世界表征与预测能力。

Reinforcement Learning of Large Language Models：大语言模型强化学习课程材料。

DSC 243: Convex Quadratics：高级优化课程中关于凸二次函数的讲义。

Pen and Paper Exercises in Machine Learning / GitHub：机器学习纸笔练习集，适合作为概念推导和考试型训练材料。

Presentations：Michael Gutmann 的统计机器学习相关 presentation 集合。

Machine Learning Visualized：机器学习概念可视化材料。

IClimbTrees：AI 与机器学习课程集合。

信号 / 雷达 / 音频

How I built a mmWave material classification radar：毫米波材料分类雷达的个人项目记录，包含硬件、信号与实验实现。

AudioMass：在线音频编辑器，适合轻量剪辑和波形处理。

计算机基础 / 理论 / 工程文章

CS-Fundamentals：面向求职准备的计算机基础资料，覆盖 DSA、计算机网络、DBMS / SQL、OOP、操作系统、系统设计与软件工程。

Respect the P v NP Problem：Computational Complexity 博客关于 P vs NP 问题的评论文章。

(Quite) A Few Words About Async：关于异步编程的长文，适合回看 async 的设计动机和复杂性来源。
