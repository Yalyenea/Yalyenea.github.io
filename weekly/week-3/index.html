macOS 技巧：Skim 高亮快捷键

为 Skim PDF 阅读器添加选中即高亮功能，模拟 Sumatra PDF 的 a 键体验：

tell application "Skim"
    activate
    tell document 1
        set theSel to (get selection)
        set theNote to make note with data theSel with properties {type:highlight note, color:deep pink}
    end tell
end tell

保存至 /Users/用户名/Library/Application Support/Skim/Scripts/highlight.scpt，然后在 系统设置 > 键盘 > 键盘快捷键 > App 快捷键 中为 Skim 添加 Ctrl+A 快捷键绑定。

注：在 mac 上使用 OrbStack 代替 Docker Desktop 时，镜像加速配置建议直接在 OrbStack 设置中添加，而非 ~/.docker/daemon.json，也可配置于 ~/.orbstack/config/docker.json
