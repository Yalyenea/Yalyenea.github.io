这周重点还是 AI Coding Agent、开发者工具链和一些高质量技术阅读。

AI Coding Agents

  • ccg-gateway: 为 Claude Code、Codex、Gemini CLI 提供网关和配置管理，适合服务商不稳定、需要频繁切换的场景。
  • Quotio: AI coding assistant 的 command center。
  • 你不知道的 Agent：原理、架构与工程实践: 一篇偏工程化的 Agent 长文。
  • autoresearch: 自动研究流程的探索，适合闭环实验类任务。
  • Harness Engineering — Agent 不好用，也许不是模型的问题: 强调 harness 设计对 Agent 可用性的决定性作用。
  • Codex use cases: OpenAI官方codex使用场景集合。
  • Understand large codebases | Codex use cases: 面向大仓库理解与改造流程的实践建议。
  • Codex Plugins: Codex 插件系统。
  • Claude Code Cheat Sheet: 常用指令速查。

Developer Tools

  • liteparse: 开源文档解析工具。
  • type4me: macOS 中文语音输入开源项目，支持识别与文本优化。
  • Zed text alignment update: 特定字符对齐功能，提升代码可读性。
  • Stitch: Google 的 AI 设计工具，偏 Figma 工作流。
  • storage.to: 文件转链接分享工具。
  • OpenCLI Hub: CLI 工具目录。
  • GitWatt: 新上线的开源项目社区。

Research & AI
