从单正弦信号估计开始

26 Apr 2025

问题模型

让我们从单复正弦信号的频率和复幅度估计开始，观察到长度为 $N$ 的数据 $\left\{ y(n) \right\}_{n=0}^{N-1}$，其满足：

$$ y(n) = \alpha \mathrm{e}^{\mathrm{j} \omega_{0} n} + w(n), \ n=0,1,\dots, N-1 $$

其中 $\alpha=a\mathrm{e}^{\mathrm{j} \phi}$ 是复幅度，$\omega_{0}$ 是待估计的角频率(Angular Frequency)，显然，我们需要从 $N$ 个复数数据中估计出三个实数未知量，即 $\boldsymbol{\theta} = [a, \phi, \omega_{0}]^{\mathrm{T}}$。一般来说需要满足 $2N \geq 3$，才能根据 $2N$ 个实数数据求出 $3$ 个未知量。 如果我们只对频率 $\omega_{0}$ 感兴趣，那么另外两个参数可以称为干扰参数(nuisance parameter)。

上式写成向量的形式为

$$ \mathbf{y} = \alpha \mathbf{s}(\omega_{0}) + \mathbf{w} \in \mathbb{C}^{N \times 1} $$

其中 $\mathbf{y} = \left[ y(0), y(1), \dots, y(N-1) \right]^{\mathrm{T}}$, $\mathbf{s}(\omega_{0}) = \left[ 1, \mathrm{e}^{\mathrm{j} \omega_{0}}, \dots, \mathrm{e}^{\mathrm{j} \omega_{0} (N-1)} \right]^{\mathrm{T}}$, $\mathbf{w} = \left[ w(0), w(1), \dots, w(N-1) \right]$。通常情况下，我们假设噪声向量服从独立同分布的零均值圆对称复高斯分布，即 $\mathbf{w} \sim \mathcal{CN}(\mathbf{0}, \sigma^{2}\mathbf{I})$ ，其中 $\sigma^{2}$ 是噪声方差，也等于噪声功率，$\mathbf{I}$ 是单位矩阵。

最大似然估计

考虑确定性对数似然函数，即假设 $\boldsymbol{\theta}$ 是已知的，则 $\alpha \mathbf{s}(\omega_{0})$ 是已知的，又假设噪声均值为零，因此数据 $\mathbf{y}$ 的均值为 $\alpha \mathbf{s}(\omega_{0})$，协方差等于噪声的协方差 $\sigma^{2} \mathbf{I}$，即服从分布 $\mathcal{CN}(\alpha \mathbf{s}(\omega_{0}),\sigma^{2}\mathbf{I})$，因此对数似然函数为：

$$ \begin{align} \mathcal{L}(\mathbf{y}|\boldsymbol{\theta}) &= \log \left[ \frac{1}{(\pi \sigma^{2})^{N}} \exp \left( - \frac{1}{\sigma^{2}} \lVert \mathbf{y} - \alpha \mathbf{s}(\omega_{0}) \rVert_{2}^{2} \right) \right] \\ &= -N\log(\pi \sigma^{2}) - \frac{1}{\sigma^{2}} \lVert \mathbf{y} - \alpha \mathbf{s}(\omega_{0}) \rVert_{2}^{2} \end{align} $$

那么最大似然估计等价于以下最小化问题

$$ \min_{\boldsymbol{\theta}} \; \lVert \mathbf{y} - \alpha \mathbf{s}(\omega_{0}) \rVert_{2}^{2} $$

显然该问题是一个非线性最小二乘(Nonlinear Least Squares, NLS)问题，但关于 $\alpha$ 是一个线性最小二乘问题，令 $f(\boldsymbol{\theta})=\lVert \mathbf{y} - \alpha \mathbf{s}(\omega_{0}) \rVert_{2}^{2}$，则

$$ \frac{ \partial f(\boldsymbol{\theta}) }{ \partial \alpha^{*} } = -\mathbf{s}^{\mathrm{H}}(\omega_{0})\left[ \mathbf{y}-\alpha \mathbf{s}(\omega_{0}) \right] $$

令$\frac{ \partial f(\boldsymbol{\theta}) }{ \partial \alpha^{*} } = -\mathbf{s}^{\mathrm{H}}(\omega_{0})\left[ \mathbf{y}-\alpha \mathbf{s}(\omega_{0}) \right]=0$，得到 $\hat{\alpha} = \frac{\mathbf{s}^{\mathrm{H}}(\omega_{0})\mathbf{y}}{\mathbf{s}^{\mathrm{H}}(\omega_{0})\mathbf{s}(\omega_{0})}=\frac{1}{N}\mathbf{s}^{\mathrm{H}}(\omega_{0})\mathbf{y}$，那么当$f(\boldsymbol{\theta})$最小时，$\alpha = \frac{\mathbf{s}^{\mathrm{H}}(\omega_{0})\mathbf{y}}{\mathbf{s}^{\mathrm{H}}(\omega_{0})\mathbf{s}(\omega_{0})}$，将其代入原问题，得到误差向量为

$$ \mathbf{e} = \mathbf{y} - \frac{\mathbf{s}(\omega_{0})\mathbf{s}^{\mathrm{H}}(\omega_{0})}{\mathbf{s}^{\mathrm{H}}(\omega_{0})\mathbf{s}(\omega_{0})}\mathbf{y} = \left( \mathbf{I} - \mathbf{P} \right) \mathbf{y} $$

其中 $\mathbf{P}=\mathbf{s}(\omega_{0})\left[ \mathbf{s}^{\mathrm{H}}(\omega_{0})\mathbf{s}(\omega_{0}) \right]^{-1}\mathbf{s}^{\mathrm{H}}(\omega_{0})$ 是到 $\mathbf{s}(\omega_{0})$ 张成的列空间上的正交投影矩阵，满足 $\mathbf{P}^{\mathrm{H}}=\mathbf{P}$，显然 $\mathbf{P}^{2}=\mathbf{P}$，是一个幂等矩阵。同样 $(\mathbf{I}-\mathbf{P})^{2}=\mathbf{I}-2\mathbf{P}+\mathbf{P}^{2}=\mathbf{I}-\mathbf{P}$。因此 $\lVert \mathbf{e} \rVert_{2}^{2}=\mathbf{e}^{\mathrm{H}}\mathbf{e}=\mathbf{y}^{\mathrm{H}}(\mathbf{I}-\mathbf{P}^{\mathrm{H}})(\mathbf{I}-\mathbf{P})\mathbf{y}=\mathbf{y}^{\mathrm{H}}(\mathbf{I}-\mathbf{P})\mathbf{y}=\mathbf{y}^{\mathrm{H}}\mathbf{y}-\mathbf{y}^{\mathrm{H}}\mathbf{P}\mathbf{y}$，第一项 $\mathbf{y}^{\mathrm{H}}\mathbf{y}$ 是一个常数，则上述优化问题等价于最大化 $\mathbf{y}^{\mathrm{H}}\mathbf{P}\mathbf{y} = \frac{\mathbf{y}^{\mathrm{H}}\mathbf{s}(\omega_{0})\mathbf{s}^{\mathrm{H}}(\omega_{0})\mathbf{y}}{\mathbf{s}^{\mathrm{H}}(\omega_{0})\mathbf{s}(\omega_{0})} = \frac{\lvert \mathbf{s}^{\mathrm{H}}(\omega_{0})\mathbf{y} \rvert^{2}}{\mathbf{s}^{\mathrm{H}}(\omega_{0})\mathbf{s}(\omega_{0})}$，很显然，分母为标量，只需要最大化 $\mathbf{s}(\omega_{0})$ 与 $\mathbf{y}$ 的内积，因此等价于离散傅里叶变换，这种方法也被称为周期图(Periodogram)法。解决该优化问题最简单的方法就是将 $\omega_{0}$ 的取值范围 $[-\pi, \pi]$ 或 $[0, 2\pi]$ 划分为较密的网格，找到取最大值时对应的 $\omega_{0}$ 即为估计值。寻找该最大值的过程可以采用 FFT 算法快速实现，一般来说 FFT 点数为 $\mathbf{y}$ 的长度 $N$，当 $N$ 较小时，分辨率较低，从而会产生离网格(Off-Grid)误差，大致可以通过以下几种方法解决：

  1. 对数据 $\mathbf{y}$ 进行补零，等价于对频域进行插值，获得较细的网格，可以借助 FFT 快速实现，但是并不会有很高的精度
  2. 在初步估计值附近取较细的网格，计算最大值，同样受限于网格的大小，且不能用 FFT 实现
  3. 取初步估计值附近的几个点，进行插值，显然插值后会更接近最大值
  4. 采用梯度上升法或者牛顿法进行迭代微调，该方法精度高，采用牛顿法收敛速度较快，但需要计算一阶和二阶导数，实际中也可以用数值微分代替直接公式求解

不同于以上的确定性似然函数，我们还可以考虑随机性对数似然函数

$$ \mathcal{L}(\mathbf{y}) = \frac{1}{\pi^{N} \det(\mathbf{C})} \exp \left\{ -\left[ \mathbf{y} - \mathbb{E}(\mathbf{y}) \right] ^{\mathrm{H}} \mathbf{C}^{-1} \left[ \mathbf{y} - \mathbb{E}(\mathbf{y}) \right] \right\} $$

通常情况下，我们假设 $\mathbf{y}$ 的均值为零，不过即使不为零，我们也可以通过对数据减去均值的操作使得数据的均值为零。

最佳线性无偏估计 / 最小均方误差估计

回到模型 $\mathbf{y} = \alpha \mathbf{s}(\omega_{0}) + \mathbf{w}$，显然 $\mathbf{y}$ 对 $\alpha$ 是线性的，那么是否存在一种最佳线性无偏估计量(Best Linear Unbiased Estimator, BLUE)呢？ 设加权系数为 $\mathbf{h} \in \mathbb{C}^{N \times 1}$，那么 $\hat{\alpha}=\mathbf{h}^{\mathrm{H}}\mathbf{y}$。根据无偏性，有：

$$ \mathbb{E}(\hat{\alpha}) = \mathbb{E}(\mathbf{h}^{\mathrm{H}}\mathbf{y}) = \mathbb{E}(\alpha \mathbf{h}^{\mathrm{H}}\mathbf{s}(\omega_{0})+\mathbf{h}^{\mathrm{H}}\mathbf{w}) = \mathbf{h}^{\mathrm{H}}\mathbf{s}(\omega_{0})\mathbb{E}(\alpha)+\mathbf{h}^{\mathrm{H}}\mathbb{E}(\mathbf{w}) = \mathbf{h}^{\mathrm{H}}\mathbf{s}(\omega_{0}) \alpha $$

令 $\mathbb{E}(\hat{\alpha})=\alpha$，有 $\mathbf{h}^{\mathrm{H}}\mathbf{s}(\omega_{0})=1$。根据最小方差准则，有：

$$ \begin{align} \operatorname{Var}(\hat{\alpha}) &= \mathbb{E}\left\{ \left| \mathbf{h}^{\mathrm{H}}\mathbf{y} - \mathbf{h}^{\mathrm{H}}\mathbb{E}(\mathbf{y}) \right| ^{2} \right\} \\ &= \mathbb{E}\left\{ \mathbf{h}^{\mathrm{H}} \left[ \mathbf{y} - \mathbb{E}(\mathbf{y}) \right] \left[ \mathbf{y} - \mathbb{E}(\mathbf{y}) \right]^{\mathrm{H}} \mathbf{h} \right\} \\ &= \mathbf{h}^{\mathrm{H}} \mathbf{C}_{\mathbf{y}\mathbf{y}} \mathbf{h} \end{align} $$

其中 $\mathbf{C}_{\mathbf{y}\mathbf{y}}=\mathbb{E}\left\{ \left[ \mathbf{y} - \mathbb{E}(\mathbf{y}) \right] \left[ \mathbf{y} - \mathbb{E}(\mathbf{y}) \right]^{\mathrm{H}} \right\}$ 称为协方差矩阵。因此最终有以下优化问题：

$$ \begin{array}{rl} \min & \mathbf{h}^{\mathrm{H}} \mathbf{C}_{\mathbf{y}\mathbf{y}} \mathbf{h} \\ \text{s.t.} & \mathbf{h}^{\mathrm{H}}\mathbf{s}(\omega_{0})=1 \end{array} $$

该问题的意义是：在保持无失真的情况下，最小化方差，均值为 $0$ 的情况下方差等于功率，该方法在 DoA 估计或 Beamforming 中通常称为最小方差无失真响应(Minimum Variance Distortionless Response, MVDR) 或者 Capon(人名) 。此问题的优化目标是二次函数，约束为线性方差，是一个凸优化问题，可以直接应用拉格朗日乘子法解决，令

$$ L(\mathbf{h}, \lambda) = \mathbf{h}^{\mathrm{H}} \mathbf{C}_{\mathbf{y}\mathbf{y}} \mathbf{h} + \lambda \left[ 1 - \mathbf{h}^{\mathrm{H}}\mathbf{s}(\omega_{0}) \right] $$

则 $\frac{ \partial L }{ \partial \mathbf{h}^{*} } = \mathbf{C}_{\mathbf{y}\mathbf{y}}\mathbf{h}-\lambda \mathbf{s}(\omega_{0})$，$\frac{ \partial L }{ \partial \lambda }=1-\mathbf{h}^{\mathrm{H}}\mathbf{s}(\omega_{0})$，令两式等于 $0$，得到

$$ \mathbf{h}_{\text{opt}} = \frac{\mathbf{C}_{\mathbf{y}\mathbf{y}}^{-1}\mathbf{s}(\omega_{0})}{\mathbf{s}^{\mathrm{H}}(\omega_{0})\mathbf{C}_{\mathbf{y}\mathbf{y}}^{-1}\mathbf{s}(\omega_{0})}, \quad \hat{\alpha} = \frac{\mathbf{s}^{\mathrm{H}}(\omega_{0})\mathbf{C}_{\mathbf{y}\mathbf{y}}^{-1}\mathbf{y}}{\mathbf{s}^{\mathrm{H}}(\omega_{0})\mathbf{C}_{\mathbf{y}\mathbf{y}}^{-1}\mathbf{s}(\omega_{0})} $$

综上，我们根据无偏和最小方差准则得到了最佳线性无偏估计，也等价于线性最小均方误差准则，因为估计量无偏时均方误差等于方差。因为上述优化问题假设 $\omega_{0}$ 已知，类似于最大似然估计中的方法，需要遍历不同的 $\omega_{0}$ 进行网格搜索，最后选择$\left| \hat{\alpha} \right|^{2}$最大的作为最终估计。

Fisher 信息矩阵和 CRB

Fisher 信息矩阵为

$$ \left[ \mathbf{I}(\boldsymbol{\theta}) \right]_{3 \times 3} = \mathbb{E}\left\{\left( \frac{ \partial \mathcal{L} }{ \partial \boldsymbol{\theta} } \right) \left( \frac{ \partial \mathcal{L} }{ \partial \boldsymbol{\theta} } \right)^{\mathrm{H}} \right\} = \begin{bmatrix} I_{a a} & I_{a \phi} & I_{a \omega_{0}} \\ I_{\phi a} & I_{\phi \phi} & I_{\phi \omega_{0}} \\ I_{\omega_{0} a} & I_{\omega_{0} \phi} & I_{\omega_{0} \omega_{0}} \end{bmatrix} $$

显然该矩阵是一个 Hermitian(共轭对称)矩阵。令 $\boldsymbol{\mu}(\boldsymbol{\theta})=\alpha \mathbf{s}(\omega_{0})$，根据 Slepian‑Bangs 公式，有：

$$ I_{ij} = 2 \Re \left\{ \frac{ \partial \boldsymbol{\mu}^{\mathrm{H}}(\boldsymbol{\theta}) }{ \partial \theta_{i} } \frac{1}{\sigma^{2}} \frac{ \partial \boldsymbol{\mu}(\boldsymbol{\theta}) }{ \partial \theta_{j} } \right\} = \frac{2}{\sigma^{2}} \Re \left\{ \frac{ \partial \boldsymbol{\mu}^{\mathrm{H}}(\boldsymbol{\theta}) }{ \partial \theta_{i} } \frac{ \partial \boldsymbol{\mu}(\boldsymbol{\theta}) }{ \partial \theta_{j} } \right\} $$

其中 $\Re\left\{ \cdot \right\}$ 表示取复数的实部。继续求出偏导数如下：

$$ \begin{align} \frac{ \partial \boldsymbol{\mu}(\boldsymbol{\theta}) }{ \partial a } &= \mathrm{e}^{j \phi}\mathbf{s}(\omega_{0}) & \frac{ \partial \boldsymbol{\mu}^{\mathrm{H}}(\boldsymbol{\theta}) }{ \partial a } &= \left[ \frac{ \partial \boldsymbol{\mu}(\boldsymbol{\theta}) }{ \partial a } \right]^{\mathrm{H}} \\ \frac{ \partial \boldsymbol{\mu}(\boldsymbol{\theta}) }{ \partial \phi } &= ja\mathrm{e}^{j \phi}\mathbf{s}(\omega_{0}) &\frac{ \partial \boldsymbol{\mu}^{\mathrm{H}}(\boldsymbol{\theta}) }{ \partial \phi } &= \left[ \frac{ \partial \boldsymbol{\mu}(\boldsymbol{\theta}) }{ \partial a } \right]^{\mathrm{H}} \\ \frac{ \partial \boldsymbol{\mu}(\boldsymbol{\theta}) }{ \partial \omega_{0} } &= ja\mathrm{e}^{j \phi} (0, 1, \dots, N-1)^{\mathrm{T}} \circ \mathbf{s}(\omega_{0}) & \frac{ \partial \boldsymbol{\mu}^{\mathrm{H}}(\boldsymbol{\theta}) }{ \partial \omega_{0} } &= \left[ \frac{ \partial \boldsymbol{\mu}(\boldsymbol{\theta}) }{ \partial a } \right]^{\mathrm{H}} \end{align} $$

结合上式，以及 $\sum_{n=0}^{N-1}n=\frac{N(N-1)}{2},\sum_{n=0}^{N-1}n^{2}=\frac{N(N-1)(2N-1)}{6}$，可以得到

$$ \mathbf{I}(\boldsymbol{\theta}) = \frac{2}{\sigma^{2}} \begin{bmatrix} N & 0 & 0 \\ 0 & N a^{2} & a^{2} \sum_{n=0}^{N-1} n \\ 0 & a^{2} \sum_{n=0}^{N-1} n & a^{2} \sum_{n=0}^{N-1}n^{2} \end{bmatrix} = \begin{bmatrix} N & 0 & 0 \\ 0 & N a^{2} & \frac{N(N-1) a^{2}}{2} \\ 0 & \frac{N(N-1) a^{2}}{2} & \frac{N(N-1)(2N-1)a^{2}}{6} \end{bmatrix} $$

根据块对角矩阵的逆$\begin{bmatrix}\mathbf{A} & \mathbf{0} \\ \mathbf{0} & \mathbf{B}\end{bmatrix}^{-1} = \begin{bmatrix}\mathbf{A}^{-1} & \mathbf{0} \\ \mathbf{0} & \mathbf{B}^{-1}\end{bmatrix}$ 以及 $\begin{bmatrix}a & b \\ c & d\end{bmatrix}^{-1}=\frac{1}{ad-bc} \begin{bmatrix} d & -b \\ -c & a \end{bmatrix}$，求得 Fisher 信息矩阵的逆为(利用 sympy 通过符号计算出矩阵逆的程序见附录)：

$$ \mathbf{I}^{-1}(\boldsymbol{\theta}) = \begin{bmatrix} \frac{\sigma^{2}}{2N} & 0 & 0 \\ 0 & \frac{(2N-1)\sigma^{2}}{N(N+1)a^{2}} & -\frac{3\sigma^{2}}{N(N+1)a^{2}} \\ 0 & -\frac{3\sigma^{2}}{N(N+1)a^{2}} & \frac{6\sigma^{2}}{N(N^{2}-1)a^{2}} \end{bmatrix} $$

从而可以得到无偏估计的 Cramér-Rao Bound(CRB) 为 $\text{CRB}(\boldsymbol{\theta})=\mathbf{I}^{-1}(\boldsymbol{\theta})$，是估计量方差的下界。

扩展至多正弦信号

$$ y(n) = \sum_{k=1}^{K} a_{k} \mathrm{e}^{\mathrm{j}(\omega_{k} n + \phi_{k})} + w(n), \quad n = 0, 1, \dots, N-1 $$

$$ \begin{bmatrix} y(0) \\ y(1) \\ \vdots \\ y(N-1) \end{bmatrix} = \begin{bmatrix} 1 & 1 & \cdots & 1 \\ \mathrm{e}^{j \omega_{1}} & \mathrm{e}^{j \omega_{2}} & \cdots & \mathrm{e}^{j \omega_{K}} \\ \vdots & \vdots & \ddots & \vdots \\ \mathrm{e}^{j (N-1) \omega_{1}} & \mathrm{e}^{j (N-1) \omega_{2}} & \cdots & \mathrm{e}^{j (N-1) \omega_{K}} \end{bmatrix} \begin{bmatrix} \alpha_{1} \\ \alpha_{2} \\ \vdots \\ \alpha_{K} \end{bmatrix} + \begin{bmatrix} w(0) \\ w(1) \\ \vdots \\ w(N-1) \end{bmatrix} $$
  • $\alpha_{k} = a_{k}\mathrm{e}^{j \phi_{k}}$
  • $\mathbf{S}(\boldsymbol{\omega}) = \begin{bmatrix}1 & 1 & \cdots & 1 \\ \mathrm{e}^{j \omega_{1}} & \mathrm{e}^{j \omega_{2}} & \cdots & \mathrm{e}^{j \omega_{K}} \\ \vdots & \vdots & \ddots & \vdots \\ \mathrm{e}^{j (N-1) \omega_{1}} & \mathrm{e}^{j (N-1) \omega_{2}} & \cdots & \mathrm{e}^{j(N-1) \omega_{K}}\end{bmatrix}$ $$ \mathbf{y} = \mathbf{S}(\boldsymbol{\omega}) \boldsymbol{\alpha} + \mathbf{w} $$

参数向量 $\boldsymbol{\theta}$ 是广义的，包含所有待估参数：幅度 $\{a_k\}$、相位 $\{\phi_k\}$、频率 $\{\omega_k\}$。

在复高斯白噪声 $\mathbf{w} \sim \mathcal{CN}(0, \sigma^2\mathbf{I})$ 的假设下，观测向量 $\mathbf{y}$ 的似然函数为：

$$ \begin{align} \mathcal{L}(\mathbf{y}|\boldsymbol{\theta}) &= \log \left[ \frac{1}{(\pi \sigma^{2})^{N}} \exp \left( - \frac{1}{\sigma^{2}} \lVert \mathbf{y} - \mathbf{S}(\boldsymbol{\omega})\boldsymbol{\alpha} \rVert_{2}^{2} \right) \right] \\ &= -N\log(\pi \sigma^{2}) - \frac{1}{\sigma^{2}} \lVert \mathbf{y} - \mathbf{S}(\boldsymbol{\omega})\boldsymbol{\alpha} \rVert_{2}^{2} \end{align} $$

考虑待估计参数向量 $\boldsymbol{\theta}=[\boldsymbol{\omega}^{\mathrm{T}}, \mathbf{a}^{\mathrm{T}},\boldsymbol{\phi}^{\mathrm{T}}]^{\mathrm{T}}$，其中 $\boldsymbol{\omega}=[\omega_{1},\omega_{2},\dots \omega_{K}]^{\mathrm{T}}, \mathbf{a}=[a_{1},a_{2},\dots,a_{K}]^{\mathrm{T}}, \boldsymbol{\phi}=[\phi_{1},\phi_{2},\dots,\phi_{K}]^{\mathrm{T}}$，以上定义基于习惯都是列向量，当然如果写成行向量的形式会更加简洁。

在复高斯观测模型下，可以证明（通过Slepian-Bangs公式）：

$$ I_{pq} = \frac{2}{\sigma^2} \Re \left\{ \left( \frac{\partial \mu}{\partial \theta_p} \right)^H \frac{\partial \mu}{\partial \theta_q} \right\} $$

其中 $\boldsymbol{\mu} = \mathbf{S}\boldsymbol{\alpha}$。

$$ \mathbf{I} = \frac{2}{\sigma^{2}} \Re\left\{ \frac{\partial\boldsymbol{\mu}^{\mathrm{H}}}{\partial \boldsymbol{\theta}} \frac{\partial \boldsymbol{\mu}}{\partial \boldsymbol{\theta}^{\mathrm{T}}} \right\} $$
  • 对幅度 $a_k$: $\frac{\partial \boldsymbol{\mu}}{\partial a_k} = e^{j\phi_k} \mathbf{s}(\omega_k)$, $\quad \frac{\partial \boldsymbol{\mu}^{\mathrm{H}}}{\partial a_k} = e^{-j\phi_k} \mathbf{s}^H(\omega_k)$
  • 对相位 $\phi_k$: $\frac{\partial \boldsymbol{\mu}}{\partial \phi_k} = j a_k e^{j\phi_k} \mathbf{s}(\omega_k) = j \alpha_k \mathbf{s}(\omega_k)$, $\quad \frac{\partial \boldsymbol{\mu}^{\mathrm{H}}}{\partial \phi_k} = -j \alpha_k^* \mathbf{s}^H(\omega_k)$
  • 对频率 $\omega_k$: $\frac{\partial \boldsymbol{\mu}}{\partial \omega_k} = \alpha_k \dot{\mathbf{s}}(\omega_k)$, $\quad \frac{\partial \boldsymbol{\mu}^{\mathrm{H}}}{\partial \omega_k} = \alpha_k^* \dot{\mathbf{s}}^H(\omega_k)$, 其中 $\dot{\mathbf{s}}(\omega_k) = \frac{\partial \mathbf{s}(\omega_k)}{\partial \omega_k} = [0, j e^{j\omega_k}, \dots, j(N-1)e^{j(N-1)\omega_k}]^T$。

将这些偏导数写成矩阵形式更紧凑。定义：

  • $\mathbf{S} = [\mathbf{s}(\omega_1), \dots, \mathbf{s}(\omega_K)]$ （$N \times K$）
  • $\dot{\mathbf{S}} = [\dot{\mathbf{s}}(\omega_1), \dots, \dot{\mathbf{s}}(\omega_K)]$ （$N \times K$）
  • $\dot{\mathbf{S}}=\mathrm{j}\operatorname{diag}([0,1,..,N-1])\mathbf{S}=\mathrm{j}\mathbf{D}_{N}\mathbf{S}$
  • $\mathbf{D}_{\phi} = \operatorname{diag}([e^{j\phi_1}, \dots, e^{j\phi_K}])$ （$K \times K$）
  • $\mathbf{D}_\alpha = \operatorname{diag}([\alpha_1, \dots, \alpha_K])$ （$K \times K$）

则偏导数矩阵为：

$$ \frac{\partial \boldsymbol{\mu}}{\partial \boldsymbol{\omega}^T} = \dot{\mathbf{S}} \mathbf{D}_\alpha, \quad \frac{\partial \boldsymbol{\mu}}{\partial \mathbf{a}^T} = \mathbf{S} \mathbf{D}_{\phi}, \quad \frac{\partial \boldsymbol{\mu}}{\partial \boldsymbol{\phi}^T} = \mathrm{j}\mathbf{S} \mathbf{D}_\alpha $$

考虑待估计参数向量 $\boldsymbol{\theta}=\left[ \boldsymbol{\omega}^{\mathrm{T}}, (\boldsymbol{\alpha}^{x})^{\mathrm{T}},(\boldsymbol{\alpha}^{x})^{\mathrm{T}} \right]^{\mathrm{T}}$，

$$ \frac{ \partial \boldsymbol{\mu} }{ \partial \boldsymbol{\omega}^{\mathrm{T}} } = \dot{\mathbf{S}}\mathbf{D}_{\alpha}, \quad \frac{ \partial \boldsymbol{\mu} }{ \partial (\boldsymbol{\alpha}^{x})^{\mathrm{T}} } = \mathbf{S}, \quad \frac{ \partial \boldsymbol{\mu} }{ \partial (\boldsymbol{\alpha}^{y})^{\mathrm{T}} } = \mathrm{j}\mathbf{S} $$

记 $\dot{\mathbf{S}}_{\alpha}=\dot{\mathbf{S}}\mathbf{D}_{\alpha}$，则

$$ \dot{\boldsymbol{\mu}} = \frac{ \partial \boldsymbol{\mu} }{ \partial \boldsymbol{\theta}^{\mathrm{T}} } = \begin{bmatrix} \dot{\mathbf{S}}_{\alpha} & \mathbf{S} & \mathrm{j}\mathbf{S} \end{bmatrix} $$

$$ \mathbf{I} = \frac{2}{\sigma^{2}}\Re(\dot{\boldsymbol{\mu}}^{\mathrm{H}} \dot{\boldsymbol{\mu}}) = \frac{2}{\sigma^{2}}\Re \left\{ \begin{bmatrix} \dot{\mathbf{S}}_{\alpha}^{\mathrm{H}} \\ \mathbf{S}^{\mathrm{H}} \\ -\mathrm{j}\mathbf{S}^{\mathrm{H}} \end{bmatrix} \begin{bmatrix} \dot{\mathbf{S}}_{\alpha} & \mathbf{S} & \mathrm{j}\mathbf{S} \end{bmatrix} \right\} $$

令 $\mathbf{F}=\begin{bmatrix}\mathbf{I} & \mathbf{0} & \mathbf{0} \\ \mathbf{X} & \mathbf{I} & \mathbf{0} \\ \mathbf{Y} & \mathbf{0} & \mathbf{I}\end{bmatrix}$，暂且假设该矩阵是实矩阵，且 $\mathbf{Z}=\mathbf{X}+\mathrm{j}\mathbf{Y}$，则

$$ \dot{\boldsymbol{\mu}} \mathbf{F} = \begin{bmatrix} \dot{\mathbf{S}}_{\alpha} & \mathbf{S} & \mathrm{j}\mathbf{S} \end{bmatrix} \begin{bmatrix}\mathbf{I} & \mathbf{0} & \mathbf{0} \\ \mathbf{X} & \mathbf{I} & \mathbf{0} \\ \mathbf{Y} & \mathbf{0} & \mathbf{I}\end{bmatrix} = \begin{bmatrix} \dot{\mathbf{S}}_{\alpha} + \mathbf{S}\mathbf{X} + \mathrm{j}\mathbf{S}\mathbf{Y} & \mathbf{S} & \mathrm{j}\mathbf{S} \end{bmatrix} = \begin{bmatrix} \dot{\mathbf{S}}_{\alpha} + \mathbf{S}\mathbf{Z} & \mathbf{S} & \mathrm{j}\mathbf{S} \end{bmatrix} $$

那么

$$ \begin{align} \mathbf{F}^{\mathrm{H}}\mathbf{I}\mathbf{F} &= \frac{2}{\sigma^{2}}\mathbf{F}^{\mathrm{H}} \Re\left( \dot{\boldsymbol{\mu}}^{\mathrm{H}} \dot{\boldsymbol{\mu}} \right) \mathbf{F} = \frac{2}{\sigma^{2}} \Re\left[ \left( \dot{\boldsymbol{\mu}} \mathbf{F} \right)^{\mathrm{H}} \dot{\boldsymbol{\mu}} \mathbf{F} \right] \\ &= \frac{2}{\sigma^{2}} \Re\left\{ \begin{bmatrix} \dot{\mathbf{S}}_{\alpha}^{\mathrm{H}} + \mathbf{Z}^{\mathrm{H}}\mathbf{S}^{\mathrm{H}} \\ \mathbf{S}^{\mathrm{H}} \\ -\mathrm{j}\mathbf{S}^{\mathrm{H}} \end{bmatrix} \begin{bmatrix} \dot{\mathbf{S}}_{\alpha} + \mathbf{S}\mathbf{Z} & \mathbf{S} & \mathrm{j}\mathbf{S} \end{bmatrix} \right\} \end{align} $$

又因为

$$ \mathbf{I}^{-1} = \mathbf{F}\mathbf{F}^{-1}\mathbf{I}^{-1}(\mathbf{F}^{\mathrm{H}})^{-1}\mathbf{F}^{\mathrm{H}} = \mathbf{F}\left( \mathbf{F}^{\mathrm{H}} \mathbf{I} \mathbf{F} \right)^{-1} \mathbf{F}^{\mathrm{H}} $$

所以想要求出矩阵的逆，最好上式中的 $\mathbf{F}^{\mathrm{H}} \mathbf{I} \mathbf{F}$ 是分块对角矩阵，即不在对角线上的矩阵都是 $\mathbf{0}$，那么有

$$ \mathbf{S}^{\mathrm{H}}(\dot{\mathbf{S}}_{\alpha} + \mathbf{S}\mathbf{Z}) = \mathbf{0} $$

可以得到

$$ \begin{align} \mathbf{Z} &= -(\mathbf{S}^{\mathrm{H}}\mathbf{S})^{-1}\mathbf{S}^{\mathrm{H}} \dot{\mathbf{S}}_{\alpha} \\ \dot{\mathbf{S}}_{\alpha} + \mathbf{S}\mathbf{Z} &= \left[ \mathbf{I} - \mathbf{S}(\mathbf{S}^{\mathrm{H}}\mathbf{S})^{-1}\mathbf{S}^{\mathrm{H}} \right] \dot{\mathbf{S}}_{\alpha} = \mathbf{P}_{\perp} \dot{\mathbf{S}}_{\alpha} \end{align} $$

其中 $\mathbf{P}_{\perp} = \mathbf{I} - \mathbf{S}(\mathbf{S}^{\mathrm{H}}\mathbf{S})^{-1}\mathbf{S}^{\mathrm{H}}$ 是投影到矩阵 $\mathbf{S}$ 列空间的正交补空间上的投影矩阵，满足 $\mathbf{P}_{\perp} = \mathbf{P}_{\perp}^{2} = \mathbf{P}_{\perp}^{\mathrm{H}}$，既是幂等矩阵又是共轭对称矩阵。将其代入 $\mathbf{F}^{\mathrm{H}}\mathbf{I}\mathbf{F}$，有

$$ \mathbf{F}^{\mathrm{H}}\mathbf{I}\mathbf{F} = \frac{2}{\sigma^{2}}\Re\left\{ \begin{bmatrix} \dot{\mathbf{S}}_{\alpha}^{\mathrm{H}} \mathbf{P}_{\perp} \dot{\mathbf{S}}_{\alpha} & \mathbf{0} & \mathbf{0} \\ \mathbf{0} & \mathbf{S}^{\mathrm{H}}\mathbf{S} & \mathrm{j}\mathbf{S}^{\mathrm{H}}\mathbf{S} \\ \mathbf{0} & -\mathrm{j}\mathbf{S}^{\mathrm{H}}\mathbf{S} & \mathbf{S}^{\mathrm{H}}\mathbf{S} \end{bmatrix} \right\} = \frac{2}{\sigma^{2}} \begin{bmatrix} \Re\left( \dot{\mathbf{S}}_{\alpha}^{\mathrm{H}} \mathbf{P}_{\perp} \dot{\mathbf{S}}_{\alpha} \right) & \mathbf{0} & \mathbf{0} \\ \mathbf{0} & \Re(\mathbf{S}^{\mathrm{H}}\mathbf{S}) & -\Im(\mathbf{S}^{\mathrm{H}}\mathbf{S}) \\ \mathbf{0} & \Im(\mathbf{S}^{\mathrm{H}}\mathbf{S}) & \Re(\mathbf{S}^{\mathrm{H}}\mathbf{S}) \end{bmatrix} $$

令 $\mathbf{Q}=\left( \mathbf{S}^{\mathrm{H}}\mathbf{S} \right)^{-1}$，根据块对角矩阵的逆以及复数分块矩阵的逆(见附录)，

$$ \left( \mathbf{F}^{\mathrm{H}}\mathbf{I}\mathbf{F} \right)^{-1} = \frac{\sigma^{2}}{2} \begin{bmatrix} \left[ \Re\left( \dot{\mathbf{S}}_{\alpha}^{\mathrm{H}} \mathbf{P}_{\perp} \dot{\mathbf{S}}_{\alpha} \right) \right]^{-1} & \mathbf{0} & \mathbf{0} \\ \mathbf{0} & \Re(\mathbf{Q}) & -\Im(\mathbf{Q}) \\ \mathbf{0} & \Im(\mathbf{Q}) & \Re(\mathbf{Q}) \end{bmatrix} $$

令 $\mathbf{C}=\left[ \Re\left( \dot{\mathbf{S}}_{\alpha}^{\mathrm{H}} \mathbf{P}_{\perp} \dot{\mathbf{S}}_{\alpha} \right) \right]^{-1}$，所以

$$ \begin{align} \mathbf{I}^{-1} &= \mathbf{F}\left( \mathbf{F}^{\mathrm{H}} \mathbf{I} \mathbf{F} \right)^{-1} \mathbf{F}^{\mathrm{H}} \\ &= \frac{\sigma^{2}}{2}\begin{bmatrix}\mathbf{I} & \mathbf{0} & \mathbf{0} \\ \mathbf{X} & \mathbf{I} & \mathbf{0} \\ \mathbf{Y} & \mathbf{0} & \mathbf{I}\end{bmatrix} \begin{bmatrix} \mathbf{C} & \mathbf{0} & \mathbf{0} \\ \mathbf{0} & \Re(\mathbf{Q}) & -\Im(\mathbf{Q}) \\ \mathbf{0} & \Im(\mathbf{Q}) & \Re(\mathbf{Q}) \end{bmatrix} \begin{bmatrix}\mathbf{I} & \mathbf{X}^{\mathrm{T}} & \mathbf{Y}^{\mathrm{T}} \\ \mathbf{0} & \mathbf{I} & \mathbf{0} \\ \mathbf{0} & \mathbf{0} & \mathbf{I}\end{bmatrix} \\ &= \frac{\sigma^{2}}{2}\begin{bmatrix} \mathbf{C} & \mathbf{0} & \mathbf{0} \\ \mathbf{X}\mathbf{C} & \Re(\mathbf{Q}) & -\Im(\mathbf{Q}) \\ \mathbf{Y}\mathbf{C} & \Im(\mathbf{Q}) & \Re(\mathbf{Q}) \end{bmatrix} \begin{bmatrix}\mathbf{I} & \mathbf{X}^{\mathrm{T}} & \mathbf{Y}^{\mathrm{T}} \\ \mathbf{0} & \mathbf{I} & \mathbf{0} \\ \mathbf{0} & \mathbf{0} & \mathbf{I}\end{bmatrix} \end{align} $$

最终得到的 Fisher 信息矩阵的逆（即 CRB 矩阵）为：

$$ \mathbf{I}^{-1} = \frac{\sigma^{2}}{2} \begin{bmatrix} \mathbf{C} & \mathbf{C}\mathbf{X}^{\mathrm{T}} & \mathbf{C}\mathbf{Y}^{\mathrm{T}} \\ \mathbf{X}\mathbf{C} & \mathbf{X}\mathbf{C}\mathbf{X}^{\mathrm{T}} + \Re(\mathbf{Q}) & \mathbf{X}\mathbf{C}\mathbf{Y}^{\mathrm{T}} - \Im(\mathbf{Q}) \\ \mathbf{Y}\mathbf{C} & \mathbf{Y}\mathbf{C}\mathbf{X}^{\mathrm{T}} + \Im(\mathbf{Q}) & \mathbf{Y}\mathbf{C}\mathbf{Y}^{\mathrm{T}} +\Re(\mathbf{Q}) \end{bmatrix} $$

其中

$$ \begin{align} \mathbf{C} &= \left[ \Re\left( \dot{\mathbf{S}}_{\alpha}^{\mathrm{H}} \mathbf{P}_{\perp} \dot{\mathbf{S}}_{\alpha} \right) \right]^{-1} \\ \mathbf{X} &= -\Re\left[ (\mathbf{S}^{\mathrm{H}}\mathbf{S})^{-1}\mathbf{S}^{\mathrm{H}} \dot{\mathbf{S}}_{\alpha} \right] \\ \mathbf{Y} &= -\Im\left[ (\mathbf{S}^{\mathrm{H}}\mathbf{S})^{-1}\mathbf{S}^{\mathrm{H}} \dot{\mathbf{S}}_{\alpha} \right] \\ \mathbf{Q} &= \left( \mathbf{S}^{\mathrm{H}}\mathbf{S} \right)^{-1} \end{align} $$

且 $\mathbf{P}_{\perp} = \mathbf{I} - \mathbf{S}(\mathbf{S}^{\mathrm{H}}\mathbf{S})^{-1}\mathbf{S}^{\mathrm{H}}, \dot{\mathbf{S}}_{\alpha}=\mathrm{j}\mathbf{D}_{N}\mathbf{S}\mathbf{D}_{\alpha}$。继续化简 $\mathbf{C}$，

$$ \begin{align} \mathbf{C} &= \left[ \Re\left( -\mathrm{j}\mathbf{D}_{\alpha}^{*}\mathbf{S}^{\mathrm{H}}\mathbf{D}_{N}\mathbf{P}_{\perp}\mathrm{j}\mathbf{D}_{N}\mathbf{S}\mathbf{D}_{\alpha} \right) \right]^{-1} \\ &= \left[ \Re\left(\mathbf{D}_{\alpha}^{*}\mathbf{S}^{\mathrm{H}}\mathbf{D}_{N}\mathbf{P}_{\perp}\mathbf{D}_{N}\mathbf{S}\mathbf{D}_{\alpha} \right) \right]^{-1} \\ &= \left\{ \Re\left[ \mathbf{D}_{\alpha}^{*}\mathbf{S}^{\mathrm{H}}\mathbf{D}_{N}\left( \mathbf{I} - \mathbf{S}(\mathbf{S}^{\mathrm{H}}\mathbf{S})^{-1}\mathbf{S}^{\mathrm{H}} \right) \mathbf{D}_{N}\mathbf{S}\mathbf{D}_{\alpha} \right] \right\}^{-1} \\ &= \left\{ \Re\left[ \mathbf{D}_{\alpha}^{*}\left( \mathbf{S}^{\mathrm{H}}\mathbf{D}_{N}^{2}\mathbf{S} - \mathbf{S}^{\mathrm{H}}\mathbf{D}_{N}\mathbf{S}(\mathbf{S}^{\mathrm{H}}\mathbf{S})^{-1}\mathbf{S}^{\mathrm{H}} \mathbf{D}_{N}\mathbf{S} \right) \mathbf{D}_{\alpha} \right] \right\}^{-1} \end{align} $$

当 $N$ 足够大时，有

$$ \begin{align} \mathbf{S}^{\mathrm{H}}\mathbf{D}_{N}^{2}\mathbf{S} &\approx \sum_{n=0}^{N-1} n^{2} \mathbf{I}_{K} = \frac{N(N-1)(2N-1)}{6}\mathbf{I}_{K} \\ \mathbf{S}^{\mathrm{H}}\mathbf{D}_{N}\mathbf{S} &\approx \sum_{n=0}^{N-1} n \mathbf{I}_{K} = \frac{N(N-1)}{2}\mathbf{I}_{K} \\ \mathbf{S}^{\mathrm{H}}\mathbf{S} &\approx N \mathbf{I}_{K}, \quad \left( \mathbf{S}^{\mathrm{H}}\mathbf{S} \right)^{-1} \approx \frac{1}{N} \mathbf{I}_{K} \\ \mathbf{X} &\approx \Re\left(\mathrm{j} \frac{N-1}{2}\mathbf{D}_{\alpha} \right) = - \Im\left( \frac{N-1}{2}\mathbf{D}_{\alpha} \right) \\ \mathbf{Y} &\approx \Im\left(\mathrm{j} \frac{N-1}{2}\mathbf{D}_{\alpha} \right) = \Re\left( \frac{N-1}{2}\mathbf{D}_{\alpha} \right) \end{align} $$

此时经过化简

$$ \begin{align} \mathbf{C} &= \left\{ \Re \left\{ \mathbf{D}_{\alpha}^{*} \left[ \frac{N(N-1)(2N-1)}{6}\mathbf{I}_{K} - \frac{N(N-1)}{2}\mathbf{I}_{K} \frac{1}{N} \mathbf{I}_{K} \frac{N(N-1)}{2}\mathbf{I}_{K} \right] \mathbf{D}_{\alpha} \right\} \right\}^{-1} \\ &= \left[ \frac{N(N-1)(N+1)}{12} \mathbf{D}_{\alpha}^{*} \mathbf{D}_{\alpha} \right]^{-1} \approx \frac{12}{N^{3}}\mathbf{D}_{|\alpha|^{2}}^{-1}, \quad \mathbf{D}_{|\alpha|^{2}} = \operatorname{diag}\left( \left[ |\alpha_{1}|^{2}, |\alpha_{2}|^{2}, \dots, |\alpha_{K}|^{2} \right] \right) \end{align} $$

所以

$$ \text{CRB}(\boldsymbol{\omega}) = \frac{\sigma^{2}}{2}\mathbf{C} \approx \frac{6 \sigma^{2}}{N^{3}}\mathbf{D}_{|\alpha|^{2}}^{-1}, \quad \text{CRB}(\omega_{k}) = \frac{6 \sigma^{2}}{N^{3} |\alpha_{k}|^{2}} = \frac{6}{N^{3} \cdot \text{SNR}_{k}} $$

同样可以化简其他对角线上的元素为：

$$ \begin{align} \text{CRB}(\boldsymbol{\alpha}^{x}) = \mathbf{X}\mathbf{C}\mathbf{X}^{\mathrm{T}} + \Re(\mathbf{Q}) &\approx \frac{3}{N}\mathbf{D}_{|\alpha^{y}|^{2}}\mathbf{D}_{|\alpha|^{2}}^{-1} + \frac{1}{N}\mathbf{I} \\ \text{CRB}(\boldsymbol{\alpha}^{y}) = \mathbf{Y}\mathbf{C}\mathbf{Y}^{\mathrm{T}} + \Re(\mathbf{Q}) &\approx \frac{3}{N}\mathbf{D}_{|\alpha^{x}|^{2}}\mathbf{D}_{|\alpha|^{2}}^{-1} + \frac{1}{N}\mathbf{I} \end{align} $$

import sympy as sp

# Define symbols
N, a, sigma = sp.symbols('N a sigma', real=True, positive=True)

# Define the components of the matrix
# Sum of n from 0 to N-1
sum_n = N*(N-1)/2
# Sum of n^2 from 0 to N-1
sum_n2 = N*(N-1)*(2*N-1)/6

# Define the matrix M (inner part)
M = sp.Matrix([
    [N, 0, 0],
    [0, N*a**2, a**2 * sum_n],
    [0, a**2 * sum_n, a**2 * sum_n2]
])

# Define the full matrix I including the scalar
I_theta = (2 / sigma**2) * M

# Calculate the inverse
I_inv = sp.simplify(I_theta.inv())

# Verify: Matrix * Inverse = Identity
verification = sp.simplify(I_theta * I_inv)

# Output results
print("Inverse Matrix:")
sp.pprint(I_inv)
print("\nVerification (should be Identity):")
sp.pprint(verification)

# Return LaTeX for the inverse
print("\nLaTeX Code:")
print(sp.latex(I_inv))
