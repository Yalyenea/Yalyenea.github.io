This week’s focus: AI Coding Agents, developer toolchains, and high-quality technical reading.

AI Coding Agents

  • ccg-gateway: Gateway and configuration management for Claude Code, Codex, Gemini CLI. Ideal for scenarios where providers are unreliable and frequent switching is needed.
  • Quotio: Command center for AI coding assistants.
  • Agents You Don’t Know: Principles, Architecture & Engineering Practice: An engineering-focused long-form article on Agents.
  • autoresearch: Automated research workflow exploration, suitable for closed-loop experimental tasks.
  • Harness Engineering — Maybe the Agent Isn’t the Model’s Problem: Emphasizes the decisive role of harness design in Agent usability.
  • Codex use cases: Official OpenAI Codex use case collection.
  • Understand large codebases: Practical advice for understanding and modifying large repositories.
  • Codex Plugins: Codex plugin system.
  • Claude Code Cheat Sheet: Quick reference for common commands.

Developer Tools

  • liteparse: Open-source document parser.

  • GitWatt: Newly launched open-source project community.

Research & AI
