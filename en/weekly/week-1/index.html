AI & Machine Learning

Models & Frameworks

  • xAI released Grok 4.2, with four Agents working simultaneously and strong search capabilities.
  • Gemini 3.1 Pro: Announcing our latest Gemini AI model — Gemini 3.1 Pro is here!
  • GitHub - fishaudio/fish-speech: SOTA Open Source TTS — Text-to-speech and voice cloning.
  • GitHub - stepfun-ai/NextStep-1 — StepFun’s autoregressive image generation model with continuous tokens (ICLR 2026 Oral).

Papers & Research

  • GitHub - MaximeVandegar/Papers-in-100-Lines-of-Code — Implementing classic AI papers in ~100 lines of code.
  • Understanding Anthropic’s Interpretability Research — A nice overview of Anthropic’s interpretability work.
  • On Surprising Effectiveness of Masking Updates in Adaptive Optimizers — A new optimizer: Momentum-aligned gradient masking (Magma).
  • GitHub - ResearAI/AutoFigure-Edit — AutoFigure: Generating publication-ready scientific illustrations (ICLR 2026).

AI Agents & Tools

  • LobsterAI — NetEase’s open-source AI Agent client.
  • GitHub - RightNow-AI/openfang — An Agent OS similar to OpenClaw.
  • GitHub - 1rgs/nanocode — Minimal Claude Code alternative in a single Python file (~250 lines, zero dependencies).
  • GitHub - minorcell/memo-code — Lightweight coding agent for terminal. Blog post discusses terminal input challenges.
  • Happy - Claude Code Mobile Client — Remote-control Claude Code on mobile (though not the most user-friendly).
  • GitHub - Panniantong/Agent-Reach — Let AI agents access social media and streaming platforms (Twitter, Reddit, YouTube, etc.).
